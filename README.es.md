# Piedra, Papel o Tijera

## Descripción
Un juego simple de Piedra, Papel o Tijera que utiliza una API y una interfaz Windows Forms. Desarrollado con fines de aprendizaje y exploración tecnológica.

## Requisitos
- C# .NET 6
- Windows Forms .NET 6.0-windows

## Cómo Ejecutar
1. Clone este repositorio
2. Ejecute la API
3. Ejecute la aplicación Forms desde `src/PedraPapelTesoura.Tela`
