# Rock, Paper, Scissors

## About
A simple Rock, Paper, Scissors game using an API and a Windows Forms UI. Built for learning purposes and exploring the technology.

## Requirements
- C# .NET 6
- Windows Forms .NET 6.0-windows

## How to Run
1. Clone this repository
2. Run the API
3. Run the Forms application from `src/PedraPapelTesoura.Tela`
